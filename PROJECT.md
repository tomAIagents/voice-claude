# VoiceClaude — Голосовой ввод для Claude Code в VSCode

## Цель проекта
Удобное, быстрое и простое приложение для голосового ввода команд в терминал VSCode,
где запущен Claude Code. Говоришь — текст появляется в строке ввода. Нажал Enter — отправил.

---

## Проблемы текущего прототипа (voice_input)
| Проблема | Описание |
|---|---|
| Лишний символ "к" | Утечка символа из горячей клавиши Ctrl+Shift+R |
| Нет визуального статуса | Непонятно — идёт запись или нет |
| Нет иконки в трее | Приложение не видно, нельзя управлять |
| Запуск через cmd | Некрасивое окно, нужно держать открытым |
| Нет настроек | Нельзя сменить хоткей, язык, модель |

---

## Требования

### Функциональные
- [ ] Push-to-talk: зажал хоткей → говоришь → отпустил → текст вставляется
- [ ] Текст вставляется точно в терминал VSCode без лишних символов
- [ ] Распознавание русского языка со знаками препинания (Whisper large-v3)
- [ ] Звуковая индикация: начало/конец записи
- [ ] Визуальная индикация: иконка в системном трее меняет вид при записи

### Нефункциональные
- [ ] Запуск одним кликом (без cmd-окна)
- [ ] Автозапуск с Windows (опционально)
- [ ] Не влияет на производительность системы в режиме ожидания
- [ ] Работает на Windows 10/11

## Стек технологий
| Компонент | Технология | Назначение |
|---|---|---|
| Распознавание речи | faster-whisper large-v3 | Офлайн, русский + пунктуация |
| Захват звука | pyaudio | Микрофон |
| Горячая клавиша | keyboard | Глобальный перехват |
| Иконка в трее | pystray + Pillow | Системный трей |
| Вставка текста | AutoHotkey v2 | Точная вставка в VSCode |
| Сборка в .exe | PyInstaller | Один файл без установки Python |

## Этапы разработки

### Этап 1 — Исправление прототипа (текущий)
- [ ] Убрать утечку символа "к"
- [ ] Переключить на push-to-talk (зажал/отпустил)
- [ ] Протестировать вставку

### Этап 1б — Предпросмотр текста
- [ ] После распознавания показывать текст во всплывающем окне
- [ ] Кнопка "Вставить" или Enter — отправить текст в терминал

### Этап 2 — Системный трей
- [ ] Иконка в трее (микрофон)
- [ ] Меню: Старт/Стоп, Настройки, Выход
- [ ] Иконка меняется при записи (анимация или цвет)

### Этап 3 — Настройки
- [ ] Файл config.json (хоткей, язык, модель)
- [ ] Окно настроек (tkinter)

### Этап 4 — Сборка
- [ ] Сборка в .exe через PyInstaller
- [ ] Автозапуск с Windows
- [ ] Финальное тестирование
