# Сессия 2026-03-25 — VoiceClaude

## Что обсуждали
Основная проблема: большая задержка при распознавании речи из-за модели large-v3 на CPU.
Обсуждали 3 варианта ускорения, выбрали вариант 2 (смена модели + убрать temp файл).
Затем доработали программу: трей, автозагрузка, окно настроек.

## Принятые решения
- Модель заменена с `large-v3` на `small` (~4x быстрее, качество достаточное для команд)
- Temp-файл убран: аудио передаётся как numpy array напрямую в faster-whisper
- Иконка в системном трее (синяя/красная/оранжевая по состоянию)
- Автозагрузка через реестр HKCU\...\Run, вкл/выкл из меню трея
- Окно настроек через кнопку ⚙ — модель, язык, хоткей, сохраняется в config.json
- Окно программы остаётся всегда видимым (не прячется в трей)
