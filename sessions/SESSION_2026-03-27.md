# Сессия 2026-03-27 — VoiceClaude

## Что обсуждали
Доработка мульти-экземплярного запуска VoiceClaude.
Эволюция решения: F-клавиши → Caps Lock везде с проверкой рабочего стола → обратно F-клавиши.

## Принятые решения
- Каждый экземпляр использует уникальную F-клавишу: F9, F10, F11, F12
- Никакого Caps Lock — убран полностью из всех конфигов
- Код проверки виртуального рабочего стола (IVirtualDesktopManager) — убран как ненужный
- `create_instance.bat` — запускать без аргументов, сам считает следующий номер
- Конфиг для нового экземпляра создаётся автоматически с правильным хоткеем

## Что сделали
- `main.py`: `_INSTANCE_HOTKEYS = ['f9', 'f10', 'f11', 'f12']` — автоназначение по INSTANCE_ID
- `main.py`: `load_config()` при автосоздании config_N.json пишет `_DEFAULT_HOTKEY` (уникальный)
- `main.py`: убраны `_get_vdm`, `is_on_current_desktop`, `_store_hwnd`, `_main_hwnd`, `_vdm_ptr`
- `config.json`: hotkey = f9 (экземпляр 0)
- `config_1.json`: hotkey = f10 (экземпляр 1)
- `create_instance.py`: шаблон VBS без синтаксических ошибок Python (Chr(34) вместо "")
- `create_instance.bat`: добавлен `chcp 65001` для корректного вывода кириллицы
